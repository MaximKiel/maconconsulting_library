При создании нового проекта заполняются следующие поля:

1. Номер проекта (поле обязательное, уникальное) в формате цифр (без "Р" на конце).
    Например: "23001", "23002"
Для квартальных отчетов особый порядок заполнения: сначала цифровой код, затем квартал, затем город.
    Например: "23000_1Q_Краснодар", "23000_3Q_Москва"

2. Сегменты рынка, по которым проводилось исследование (поле обязательное). Можно выбрать одно или несколько значений
из перечня (чтобы выбрать несколько, зажмите клавишу Control/Command). Сам перечень можно редактировать, если перейти
по ссылке под полем.

3. Тип отчета (поле обязательное). Можно выбрать единственное значение из перечня. Сам перечень можно редактировать,
если перейти по ссылке под полем.

4. Форматы отчета (поле обязательное). Можно выбрать одно или несколько значений из перечня (чтобы выбрать несколько,
зажмите клавишу Control/Command). Сам перечень можно редактировать, если перейти по ссылке под полем.

5. Ключевые слова, полезные метки, характеризующие какие-либо отдельные полезные составляющие отчетов.
Можно выбрать одно или несколько значений из перечня (чтобы выбрать несколько, зажмите клавишу Control/Command).
Сам перечень можно редактировать, если перейти по ссылке под полем.

6. Год проекта (поле обязательное) в формате 4 цифр.
    Например: "2023", "2022"

7. Актуальность данных в формате "месяц.год".
    Например: "12.2023"

8. Название проекта (поле обязательное, уникальное) в текстовом формате. Название полностью повторяет название
проектной папки на сервере.
    Например: "23003Р. Ялта. МИ МЖС. Knight Frank", "23001Р. Ташкент. МИ. МЖС. CMWP"

9. Юрлицо и/или другое обозначение заказчика (поле обязательное) в текстовом формате. Важно! Название организации записывать
без организационно-правовой формы и без кавычек!
    Например: Интеркомп
Если помимо официального названия существует общепринятое, его следует записывать через запятую в этом же поле.
Аналогично следует поступать, если заказчиков несколько.
    Например: Интеркомп, СберРешения

10. Локация (поле обязательное) в текстовом формате. Должно включать обозначение страны, региона и города/муниципального
образования. Если стран/регионов/городов несколько, следует записывать их через запятую, соблюдая описанный выше порядок.
Рекомендуется записывать полное официальное название региона/города/муниципального образования для унификации данных.
    Например: "Россия, Краснодарский край, Сочи"
Если исследование проводилось в небольшом населенном пункте, который является частью другого муниципального образования,
необходимо записать оба наименования через запятую.
    Например: "Россия, Ростовская область, Краснодарский край, Ростов-на-Дону, Краснодар"
ВАЖНО: перед названием населенного пункта НЕ ставится обозначение его типа: город, село, пгт и др.
    Например: НЕПРАВИЛЬНО - "Россия, Краснодарский край, г-к. Геленджик, с. Дивноморское"
              ПРАВИЛЬНО - "Россия, Краснодарский край, Геленджик, Дивноморское"

После заполнения всех полей необходимо нажать кнопку "Добавить" внизу страницы!
